# 条件命中率诊断方法

回测前必做：检查每个条件的独立命中率，而非直接跑全量回测。

## 核心思路

完整回测之前，先用小数据集（7天 5m 数据）检查：

1. **各周期条件独立命中率** — 哪个周期是瓶颈？
2. **多周期共振率** — 同时满足所有条件的概率是多少？
3. **瓶颈定位** — 命中率 < 3% 的条件需要调整
4. **断裂原因分布** — 方向消失时，是哪个条件先掉的？

## 诊断脚本模式

```python
# Step 1: 各周期独立统计
for name, df in [('1H', df1), ('4H', df4), ('12H', df12)]:
    total = len(df.dropna())
    state_ok = (df['macd_golden'] & df['macd_above_zero']).sum()
    print(f'{name}: {state_ok}根 ({state_ok/total*100:.1f}%)')

# Step 2: 5m 粒度三周期共振率
for ts in df5.index:

# Step 3: 断裂原因分布（关键！）

```

## 实际数据案例

### 案例 1: BTC 三周期共振（2026年5月）

```
周期独立命中率（金叉状态 + 零轴上方）：
  1H:  252/701 根 (35.9%)
  4H:  138/341 根 (40.5%)
  12H: 67/161 根 (41.6%)

→ 看似每个周期都有 35-42% 命中率
  但 5m 粒度三周期共振 = 0.0%
  独立命中率 ≠ 共振率，周期时间窗口错开
```

### 案例 2: 六条件 AND 门断裂分析（BTC 60天）

```
大方向定义 = 1H金叉+零轴 + 4H金叉+零轴 + 12H金叉 + Boll向上

多头状态：仅占 11% 的 15m bar（652/5760）

当方向断裂时（15835次），各条件掉线分布：
  12h_golden:  18%  ← 无单一瓶颈
  1h_golden:   17%  ← 
  4h_golden:   17%  ← 
  1h_zero:     17%  ← 所有条件都差不多
  boll:        17%  ← 
  4h_zero:     14%  ← 

结论：六个 AND 条件均匀断裂，无法通过修一个条件显著改善。
      这是结构性多条件 AND 门问题，不是指标选择问题。
      修复路径：降维（减少条件数）或接受 11% 的可用时间窗口。
```

### 案例 3: 15m vs 1H 出场对比

```
v3（15m MACD 出场）:
  胜率 27.9%, 累计 +1.41%, 均持仓 1.5h
  出场原因: 15m死叉占 58%, 平均单笔 -0.30%
  问题: 15m 太敏感, 趋势中的微回调反复触发砍仓

v4（1H MACD 出场）:
  胜率 31.2%, 累计 +3.34%, 均持仓 5.0h
  出场原因: dir_change 占 87%, 平均单笔 +0.28%
  改善: 持仓更长, 累计翻倍, 均盈从 0.89% → 1.33%
```

## 结论

- 独立命中率 ≠ 共振率，40% × 40% × 40% ≠ 6.4%
- 六条件 AND 门导致可用时间窗口仅 11%
- 出场周期必须比入场周期大（15m 进 → 1H 出，不能同级）
- 先诊断后回测，不等完整结果才知道问题
- 一刀一刀改，一个版本只改一个变量；差了立刻回滚到已知最优版

### 案例 4: 砍 12H 的效果（BTC 60天 v4→v6）

```
v4（三重共振 1H/4H/12H）:
  32 笔, 胜率 31.2%, 累计 +3.34%

v6（1H主 + 4H确认, 砍12H）:
  54 笔, 胜率 37.0%, 累计 +7.67%

变化: 交易数 +69%, 累计翻倍
原因: 12H 零轴条件让三周期共振概率归零
      12H 本身慢，MACD 金叉状态频繁与 1H/4H 错开
      砍掉后大方向持续时间从 11% 翻倍到可用水平
```

### 案例 5: v7 失败原因（方向=None→做空 + 15m出场）

```
v7（方向消失切做空 + 15m死叉+空头排列出场）:
  36 笔, 胜率 25%, 累计 -11.27%
  17/36 笔被止损（47%）, 做空 10 笔胜率 10%

根因：
  ① 方向=None 做空 = 在震荡区猜方向，BTC 震荡≠反转
  ② 15m 死叉+空头排列出场太慢，价格跌穿 1% 才能确认
  ③ 止损吃掉了做多 15m_dead 赚的 +11.73%

教训：方向=None 不交易（宁可错过不可做错）
      测试失败立刻回滚（Ian: "改回上个版本"）
```
