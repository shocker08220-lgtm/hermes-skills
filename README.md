# ianoxc/hermes-skills

Hermes Agent 技能 Tap 仓库（ianoxc 维护）。

专注 **Web3 研究效率工具**，避免与 Hermes 官方 bundled 技能重复。

## 安装

```bash
# 添加 Tap
hermes skills tap add ianoxc/hermes-skills

# 搜索
hermes skills search web3
hermes skills search polymarket
hermes skills search backtest

# 安装示例
hermes skills install ianoxc/hermes-skills/research/web3-research-toolkit
hermes skills install ianoxc/hermes-skills/research/polymarket
```

## 已收录技能

### research（Web3 研究工具）
- `web3-research-toolkit` — Web3 研究必备工具合集（Dune、KOL 追踪、合约发现、内容提取等）
- `crypto-trading-backtest` — 加密货币永续合约回测框架（含诊断优先方法论）
- `polymarket` — Polymarket 市场数据查询与分析
